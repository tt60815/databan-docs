# Databan 改版摘要

> 2026-03-12 ~ 2026-03-13
 
把掃描出來的最佳策略，接上 IB 模擬帳戶，實現**全自動即時交易**。

```
使用者在網頁勾選策略 → 存入 DB → 系統 B 自動讀取
                                      ↓
                            即時行情 → K棒聚合 → 指標計算
                                      ↓
                            比對策略條件 → 觸發進出場信號
                                      ↓
                            自動下單到 IB 模擬帳戶
                                      ↓
                            Dashboard 即時顯示結果
```

---

## 新增功能

| # | 功能 | 說明 | 日期 |
|---|------|------|------|
| 1 | 一鍵加入模擬倉 | 掃描結果頁面勾選策略，直接送入模擬倉 | 3/12 |
| 2 | 即時 K 棒聚合 | Databento 即時行情自動聚合成 5m/15m/30m/1h K 棒 | 3/12 |
| 3 | 即時指標計算 | 62 個技術指標欄位，與回測系統完全一致 | 3/12 + 3/13 |
| 4 | 策略自動評估 | 每根 K 棒到達時自動判斷是否進出場 | 3/12 + 3/13 |
| 5 | IB 自動下單 | 信號觸發後自動送單到 IB Gateway | 3/12 |
| 6 | 即時 Dashboard | 策略狀態、信號紀錄、持倉、交易紀錄一目了然 | 3/12 |
| 7 | 持倉時長統計 | 掃描結果新增平均/最大/最小持倉 K 棒數 | 3/12 |
